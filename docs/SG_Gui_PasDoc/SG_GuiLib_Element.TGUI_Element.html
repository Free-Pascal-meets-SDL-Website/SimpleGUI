procedure ctrl_MouseEnter(); virtual;

There are two types of events: - Ctrl events which are used by the controls themselves - these should only really be overwritten by new or modified widgets. Even then, the should still call the inherited procedures. Some (like setting) the mouseentered variable in ctrl_MouseEnter() are particularly important. - Normal events which are the ones that should be overwritten on a per control basis. The Ctrl events call the necessary normal ones.
