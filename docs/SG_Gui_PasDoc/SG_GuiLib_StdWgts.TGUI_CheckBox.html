Unit

SG_GuiLib_StdWgts

Declaration

type TGUI_CheckBox = object(TGUI_Canvas)

Description

:Generic Checkbox Object. Click to change its value. Can also act as a radio button by changing the ExcGroup to be less then 0 (and the same as the other now radio buttons). It only looks through one level of parent for siblings of a CheckBox type.
